# Web AOA Frontend

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

Or use docker:

```
docker-compose up --build
```

The website is available under http://localhost:8080/

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```
